Event notes

A few things we keep consistent from night to night.

We use a small, repeatable setup so the room stays comfortable for regular cafe guests and first-time visitors alike.

01

Early doors, calm start

Doors open at least 30 minutes before each event so guests can find a seat, order a drink, and settle in before the first set or demo begins.

02

Clear RSVP expectations

If a night has a seat limit or reserved tables, we say so clearly on the page and in the confirmation response to avoid confusion at the door.

03

Cafe service stays open

Espresso, tea, and a focused pastry menu remain available throughout the night, so events feel like an extension of the cafe rather than a complete reset.
